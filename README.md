# Smart Healthcare API

- Patient management
- Doctor scheduling
- Appointment booking
- Medical records system
- JWT authentication
- Swagger documentation

- Java 17
- Spring Boot 3.5
- Spring Security
- MySQL 8
- Swagger/OpenAPI 3.0
- Maven
